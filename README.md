# 👻 سكربت GHOST SIDE-BY-SIDE MERGER v2

أداة "شبحية" مبرمجة بالبايثون عشان تدمج الفيديوهات جنب بعض (Side-by-Side) أوتوماتيك من غير تدخل منك. السكربت ده معمول مخصوص لصناع المحتوى عشان يظبط فيديوهات الرياكشنات (Reaction Videos) لـ تيك توك ويوتيوب شورتس في ثواني، وكل ده من جوه Termux أو أي تيرمينال!

## 🔥 ليه السكربت ده عفريت؟ (المميزات)

* **تخليص بالجملة (Batch Processing):** ارمي أي عدد فيديوهات في الفولدر، والسكربت هيمسكهم يدمجهم كلهم ورا بعض من غير ما توجع دماغك.
* **أبعاد ذكية (بدون مط أو قص):** السكربت بيظبط مقاسات الفيديوهات أوتوماتيك (540x1920 لكل نص) وبيضيف حواف سودا شيك عشان يحافظ على جودة وأبعاد الفيديو الأصلي.
* **واجهة رايقة للـ Termux:** ألوان متناسقة، شريط بيوريك هو وصل لفين (Progress)، ومفيش أي دوشة من رسايل FFmpeg المزعجة عشان شاشتك تفضل نظيفة.
* **رندرة طلقة:** بنستخدم إعداد `-preset ultrafast` عشان نطلعلك الفيديوهات المدمجة جاهزة للرفع في أسرع وقت ممكن.

## ⚙️ محتاج إيه عشان يشتغل؟

عشان تشغل السكربت ده ويديك أداء الأشباح، لازم تتأكد إنك مسطب الحاجتين دول:

1. **Python 3**
2. **FFmpeg** (ده المحرك الأساسي اللي بيفرم الفيديوهات).
   * عشان تسطبه على **Termux**، اكتب الأمر ده: 
     ```bash
     pkg install ffmpeg
     ```
   * عشان تسطبه على **Linux**، اكتب: 
     ```bash
     sudo apt install ffmpeg
     ```

## 📂 رصّة الملفات (Folder Structure)

قبل ما تدوس إنتر وتعيش، اتأكد إن ملفاتك مترتبة في الفولدر بتاعك بالشكل ده:

```text
📁 Project_Folder/
│── 📄 script.py             (ملف البايثون بتاعنا)
│── 🎬 reaction.mp4          (فيديو الرياكشن الثابت بتاعك اللي هيتحط تحت)
│── 📁 top_parts/            (الفولدر اللي هترمي فيه كل الفيديوهات اللي هترد عليها)
│── 📁 final_side_by_side/   (الفولدر ده السكربت هيكريته لوحده ويرميلك فيه الشغل الجاهز)
